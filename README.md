# 🧩 Mini Website Builder (Rekaz Frontend Challenge)

A simple **drag & drop website builder** built with **Next.js + Zustand + dnd-kit**.  
Users can add, edit, reorder, and delete website sections with live preview.

## ✨ Features

### 🧱 Section Library
- Add pre-built sections (Header, Hero, Footer)
- One-click insertion into page

### 👀 Live Preview
- Real-time preview of the website
- Click to select sections

### ✏️ Property Editor
- Edit section title
- Edit hero description
- Delete sections

### 🔀 Drag & Drop
- Reorder sections using `dnd-kit`
- Smooth reordering experience

### 💾 State Management
- Built using **Zustand**
- Centralized state for sections and selection

### 📱 Responsive UI
- Works on mobile, tablet, and desktop

---

## 🛠️ Tech Stack

- Next.js

- Zustand
